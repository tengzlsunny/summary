## 垂直居中

代码
```
    <div class="three">
        <p>11111111111111</p>
    </div>
```
1.方法一
```
    width: fit-content; height: fit-content不能省略
    <style>
        .three {
            width: fit-content;
            height: fit-content;
            background: rebeccapurple;
            position: fixed;
            top: 0;
            left: 0;
            right: 0;
            bottom: 0;
            margin: auto;
        }
        .three p {
            margin: 20px;
        }
    </style>
```

2.方法2
```
    <style>
         body {
          width: 100%;
          height: 100%;
        }
        .three {
            background: rebeccapurple;
            position: fixed;
            top: 50%;
            left: 50%;
            transform: translate(-50%,-50%);
        }
        .three p {
            margin: 20px;
        }
    </style>
```

3.方法3
```
    <style>
        .three {
            width: 200px;
            height: 50px;
            background: rebeccapurple;
            position: fixed;
            top: 50%;
            left: 50%;
            margin-left: -100px;
            margin-top: -25px;
        }
        .three p {
            text-align: center;
            height: 50px;
            line-height: 15px;
        }
    </style>
```
